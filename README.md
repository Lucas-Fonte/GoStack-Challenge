## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install foobar.

```bash
cd backend
yarn
yarn sequelize db:migrate
yarn start
```

```bash
cd frontend
yarn 
yarn start
```

```bash
cd mobile
yarn 
yarn start
```
