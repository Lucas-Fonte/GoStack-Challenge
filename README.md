## Installation

Use um package manager, [yarn] ou [npm] , para instalar as dependências.

```bash
cd backend
yarn
yarn sequelize db:migrate
yarn start
```

```bash
cd frontend
yarn 
yarn start
```

```bash
cd mobile
yarn 
yarn start
```
